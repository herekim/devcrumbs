# 자바스크립트 런타임이란?

- 자바스크립트의 실행 환경
- 브라우저와 노드가 있음
- 크롬의 V8엔진을 활용해 만든 노드 환경에서 자바스크립트를 실행시켜준다
- 노드는 브라우저 내부에서가 아니라 OS에서 컴퓨터를 제어할 수 있다
- 99%는 서버를 만들 때 쓰인다
